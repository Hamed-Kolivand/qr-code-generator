# qr-code-generator
 QR Code Generator
یه ابزار خط فرمان ساده با Node.js که هر متنی رو به QR Code تبدیل می‌کنه و توی یه فایل ذخیره می‌کنه.

🚀 چطور اجراش کنی؟ #
bash
git clone https://github.com/Hamed-Kolivand/QR-Code-Generator
cd QR-Code-Generator
npm install
node index.js
🛠️ تکنولوژی‌ه #
Node.js
qr-image (برای تولید QR)
fs (برای ذخیره‌سازی فایل)
inquirer (برای دریافت ورودی از کاربر)
